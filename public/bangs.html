Bangs

This is like duckduckgo's bang system, however using this place instead of ddg.
This will fallback onto duckduckgo if either the bang isn't recognized or there is no bang.

How to use:
  • General use: Go to https://trinkey.web.app/bangs/?q= with the q parameter set to this format: "!{{BANG}} {{SEARCH_QUERY}}" (with all spaces either URL encoded or replaced with a plus)
  • Easy use on chrome/chrome based browsers: go to chrome://settings/searchEngines (or opera://, about://, etc.), click on the "Add" button in the "Site search" section, set the name and shortcut to anything, and then in the "URL" section, put "https://trinkey.web.app/bangs/?q=%s"
  • On firefox: right click here:
    then click the Add a Keyword for this Search... button. Name it anything, put it anywhere on the bookmarks bar, and then put something in the keyword section which will be used to easily access it from the search bar. This can be changed later by editing the bookmark.
  • On mobile: idfk look it up
